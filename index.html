How-to use the INSPIRE themes icons ?

Load the CSS

To use the icons, load the css in your HTML pages: 

      <link rel="stylesheet" href="css/inspire-themes.css"/>
To have all EU languages labels, use the following: 

      <link rel="stylesheet" href="css/inspire-themes.css"/>
      <link rel="stylesheet" href="css/inspire-themes-i18n.css"/>

Use the icons

Icon only



       <span class="iti-ac"></span>

Icon with border



     <div class="iti-box ad">
       <div class="icon"><span class="iti-ad"></span></div>
     </div>

Icon with border and acronym



     <div class="iti-box acr ad">
       <div class="icon"><span class="iti-ad"></span></div>
       <p class="label"/>
     </div>

Icon with border and label in english


Use the "ad-en" class to add the label in english. 

     <div class="iti-box full ad ad-en">
       <div class="icon"><span class="iti-ad"></span></div>
       <p class="label"/>
     </div>

Change the language code to display label in another language. eg. "en-nl"
